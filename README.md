# 📦 Instalasi

Clone repository

```bash
git clone https://github.com/spyschools/osint-indonesia.git
```

Masuk ke folder project

```bash
cd osint-indonesia
```

Berikan permission

```bash
chmod +x osint.py
```

---

# 🚀 Penggunaan

## Mencari informasi dari NIK

```bash
python3 osint.py 3275124308050003
```

---

## Mencari informasi dari Nomor HP

```bash
python3 osint.py +6281234567890
```
